## 部署 5 步走

```bash
cd C:\Users\16084\regret-mart
git init && git add . && git commit -m "feat: regret-mart"
gh repo create regret-mart --public --push --source .
vercel link
vercel --prod
```

> 不想用 CLI 也可以：在 Vercel 控制台 Import GitHub 仓库即可。
> 第一次部署可以不配置任何环境变量，会使用本地兜底模板 + 内存存储。

## 评审亮点话术（供讲解用）

- **创新性**：把市场抽象到极简——商品是情绪，价格是痛感，疗效是 AI 写的开解话。
- **可玩性**：2 分钟主流程闭环；不注册、不登录；进入即送 100 释怀币。
- **完整性**：买卖双向、定价、钱包、收藏、内容审核、降级方案、隐私声明全有。
- **合规性**：明确"虚拟币无现实价值"、关键词黑名单、心理援助热线、未成年提示。
- **技术深度**：Next.js 15 全栈 + AI 流式生成 + framer-motion 翻牌 + 静默降级到本地模板。

## 风险与回退（评委可能问）

| 问 | 答 |
|---|---|
| LLM 挂了怎么办？ | 自动降级到 3 套本地兜底模板，体验无损 |
| 数据丢失怎么办？ | 冷启动会重置，但自带 12 颗精写种子；要持久化在 README 有 Supabase 接入方案 |
| 内容审核怎么做？ | 关键词黑名单 + 长度限制 + LLM Prompt 二次审核；自伤敏感词触发心理援助提示 |
| 释怀币会被滥用吗？ | 全站明确标注"不可充值不可提现"，匿名身份每个浏览器只送 100 |
